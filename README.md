# Antigravity Skills Navigation

A premium visual navigator for the Antigravity Awesome Skills library.

## Features
- **Visual Classification**: Skills are categorized into Mobile, Web, AI, Security, etc.
- **Detailed View**: Click any skill to see its ID and detailed description.
- **Search & Filter**: Quickly find the skill you need.
- **Premium Design**: Dark mode with glassmorphism and smooth animations.
- **Data Driven**: Built from a JSON index extracted directly from the skills library.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Update Data** (Optional, if skills repository changes):
   ```bash
   node scripts/extract.js
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

## Technology Stack
- Vue 3 (Composition API)
- TypeScript
- Vite
- Vanilla CSS (Premium Aesthetics)
